# Elmos reservations backend

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Documentation
To see openapi documentation:
http://localhost:3000/api/


## Database
Rename example.env to .env and fill in the database options. In the folder database is a dump of this database.
